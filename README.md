# Boutique Maria Rosa

Site institucional desenvolvido como trabalho acadêmico de Front-End para a
Boutique Maria Rosa, uma loja de moda feminina, masculina e infantil. O projeto
apresenta o catálogo de produtos organizado por categorias e as informações da
loja.

🔗 **Acesse o site:** https://afonsolou.github.io/Loja-Maria-rosa/

## Sobre o projeto

O site reúne a vitrine de produtos da boutique, separada por público, além de uma
página com a história da loja, sua localização e contato. O objetivo é simular a
presença online de uma loja real, com navegação clara entre as seções.

## Páginas

- **Home** (`index.html`) — vitrine com novidades, conjuntos e calçados em destaque.
- **Produtos femininos** (`página2.html`) — catálogo de roupas femininas.
- **Produtos masculinos** (`página3.html`) — catálogo de roupas masculinas.
- **Produtos infantis** (`página4.html`) — catálogo de roupas infantis.
- **Sobre a loja** (`página5.html`) — história da boutique, localização (mapa) e contato.

## Tecnologias

- **HTML** — estrutura e conteúdo semântico das páginas
- **CSS** — estilização, layout em grid e efeitos visuais
- **JavaScript** — interatividade

## Estrutura de arquivos

```
.
├── index.html        # Home (vitrine)
├── página2.html      # Produtos femininos
├── página3.html      # Produtos masculinos
├── página4.html      # Produtos infantis
├── página5.html      # Sobre a loja
├── style.css         # Estilos do site
└── img/              # Imagens dos produtos e da loja
```

## Como executar localmente

Por ser um site estático, não há etapa de build. Basta clonar ou baixar o
repositório e abrir o arquivo `index.html` no navegador.

## Observação

As imagens de produto utilizadas são provisórias, obtidas de bancos de imagem
gratuitos (Pexels), servindo apenas para demonstração do layout.
